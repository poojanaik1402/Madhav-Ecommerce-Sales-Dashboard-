# 📊 Madhav Ecommerce Sales Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI Ecommerce Sales Dashboard** that helps analyze sales performance across multiple business dimensions including customers, states, product categories, payment methods, and monthly profits.

The dashboard enables users to monitor key business metrics, identify high-performing categories, and gain actionable insights for better decision-making.

---

## 🚀 Dashboard Features

- Interactive Quarter Filter
- State-wise Sales Analysis
- Category-wise Quantity Distribution
- Customer-wise Sales Performance
- Payment Mode Analysis
- Monthly Profit Trend
- Profit by Sub-Category
- KPI Cards for quick business overview

---

## 📈 Key Performance Indicators (KPIs)

- 💰 Total Sales Amount: **438K**
- 📦 Total Quantity Sold: **5615**
- 📊 Total Profit: **37K**
- 💵 Average Order Value (AOV): **121K**

---

## 📊 Dashboard Insights

- Clothing contributes the highest sales quantity (63%).
- COD is the most preferred payment method.
- Maharashtra generates the highest sales amount.
- Printers contribute the highest profit among all sub-categories.
- Monthly profit varies significantly across the year, indicating seasonal trends.
